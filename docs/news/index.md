## RandomWalker 0.3.0

CRAN release: 2025-05-06

### Breaking Changes

1.  Fix
    [\#107](https://github.com/spsanderson/RandomWalker/issues/107) -
    This change allows for the generation of random walks with up to 3
    dimensions. Due to this what was the `x` column is now called
    `step_number` for all random walk functions including rw30(). The
    `x` column is now the first dimension of a 2D/3D random walk.

### New Features

1.  Fix
    [\#105](https://github.com/spsanderson/RandomWalker/issues/105) -
    Add internal function
    [`rand_walk_column_names()`](https://www.spsanderson.com/RandomWalker/reference/rand_walk_column_names.md)
    to generate column names for random walks.
2.  Fix
    [\#142](https://github.com/spsanderson/RandomWalker/issues/142) -
    Add vector function
    [`confidence_interval()`](https://www.spsanderson.com/RandomWalker/reference/confidence_interval.md)
    to generate confidence interval tibble.
3.  Fix [\#71](https://github.com/spsanderson/RandomWalker/issues/71) -
    Add function subset_walks() to subset random walks by max or min
    value.

### Minor Fixes and Improvements

1.  Fix
    [\#107](https://github.com/spsanderson/RandomWalker/issues/107) -
    Add `.dimensions` parameter to random walk functions to allow for
    the generation of random walks with up to 3 dimensions!
